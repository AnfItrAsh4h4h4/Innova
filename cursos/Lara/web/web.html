Desenvolvimento Web

O desenvolvimento web envolve a criação de sites e aplicações usando tecnologias como HTML, CSS e JavaScript. Aqui você aprenderá como essas linguagens funcionam juntas para construir páginas modernas, bonitas e responsivas.

HTML

Estrutura do site. Define títulos, textos, imagens, botões e muito mais.

CSS

Responsável pelo visual: cores, tamanhos, animações, layouts e efeitos.

JavaScript

Dá vida ao site: interatividade, lógica, animações e funcionalidades.
